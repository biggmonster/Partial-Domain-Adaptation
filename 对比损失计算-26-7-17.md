# 标签引导的源域增强对比预训练方案

## 方案概述

预训练阶段仅使用带标签源域样本：

\[
\boxed{
L_{\mathrm{pre}}
=
L_{\mathrm{RE}}
+
\lambda_{\mathrm{CON}}L_{\mathrm{CON}}
}
\]

其中：

- \(L_{\mathrm{RE}}\) 为源域标准交叉熵损失；
- \(L_{\mathrm{CON}}\) 为标签引导的增强对比损失；
- 仅同一样本的原图与增强图互为正样本；
- 其他同类别样本保持中性；
- 仅不同类别样本及其增强视图作为负样本；
- 不进行类别频率加权，仅采用 \(0/1\) 类别掩码。

## 特征与相似度定义

设源域批次为：

\[
\mathcal B_s=\{(x_i,y_i)\}_{i=1}^{N}.
\]

对每个样本生成一个增强视图：

\[
\tilde x_i=T(x_i),
\]

并通过特征提取器得到：

\[
z_i=F(x_i),\qquad
\tilde z_i=F(\tilde x_i).
\]

余弦相似度定义为：

\[
\boxed{
s(a,b)
=
\frac{a^\mathrm Tb}
{\|a\|_2\|b\|_2}
}
\]

实际计算时应在范数中加入极小量 \(\varepsilon\)，避免零向量导致除零：

\[
s(a,b)
=
\frac{a^\mathrm Tb}
{\max(\|a\|_2\|b\|_2,\varepsilon)}.
\]

## 类别掩码与负样本集合

定义类别掩码：

\[
\boxed{
m_{ij}
=
\mathbb I(j\neq i)\mathbb I(y_j\neq y_i)
}
\]

即：

\[
m_{ij}=
\begin{cases}
1,&y_j\neq y_i,\\
0,&y_j=y_i.
\end{cases}
\]

对于锚点 \(z_i\)，负样本集合为：

\[
\boxed{
\mathcal N_i
=
\left\{
z_j,\tilde z_j
\mid y_j\neq y_i
\right\}.
}
\]

因此：

- \(\tilde z_i\) 是 \(z_i\) 的正样本；
- \(z_j,\tilde z_j\) 在 \(y_j\neq y_i\) 时是负样本；
- \(z_j,\tilde z_j\) 在 \(y_j=y_i,\ j\neq i\) 时不参与该锚点的对比计算。

## 对称对比损失

### 原图作为锚点

正样本相似度为：

\[
s_i^+
=
s(z_i,\tilde z_i).
\]

归一化分母为：

\[
\boxed{
D_i^x
=
\exp\left(\frac{s_i^+}{\tau}\right)
+
\sum_{j=1}^{N}m_{ij}
\left[
\exp\left(\frac{s(z_i,z_j)}{\tau}\right)
+
\exp\left(\frac{s(z_i,\tilde z_j)}{\tau}\right)
\right].
}
\]

损失为：

\[
\boxed{
\ell_i^{x\rightarrow\tilde x}
=
-\log
\frac{
\exp\left(s_i^+/\tau\right)
}{
D_i^x
}.
}
\]

### 增强图作为锚点

归一化分母为：

\[
\boxed{
D_i^{\tilde x}
=
\exp\left(\frac{s(\tilde z_i,z_i)}{\tau}\right)
+
\sum_{j=1}^{N}m_{ij}
\left[
\exp\left(\frac{s(\tilde z_i,z_j)}{\tau}\right)
+
\exp\left(\frac{s(\tilde z_i,\tilde z_j)}{\tau}\right)
\right].
}
\]

损失为：

\[
\boxed{
\ell_i^{\tilde x\rightarrow x}
=
-\log
\frac{
\exp\left(s(\tilde z_i,z_i)/\tau\right)
}{
D_i^{\tilde x}
}.
}
\]

### 批次损失

设有效锚点集合为：

\[
\mathcal I
=
\left\{
i\mid
\sum_{j=1}^{N}m_{ij}>0
\right\},
\]

则：

\[
\boxed{
L_{\mathrm{CON}}
=
\frac{1}{|\mathcal I|}
\sum_{i\in\mathcal I}
\frac12
\left(
\ell_i^{x\rightarrow\tilde x}
+
\ell_i^{\tilde x\rightarrow x}
\right).
}
\]

## 等价的相对相似度公式

原图方向可以等价改写为：

\[
\boxed{
\ell_i^{x\rightarrow\tilde x}
=
\log
\left[
1+
\sum_{j=1}^{N}m_{ij}
\left(
\exp\frac{s(z_i,z_j)-s_i^+}{\tau}
+
\exp\frac{s(z_i,\tilde z_j)-s_i^+}{\tau}
\right)
\right].
}
\]

该形式更直观地体现优化目标：

\[
\boxed{
s(z_i,\tilde z_i)
>
s(z_i,z_j),\quad
s(z_i,\tilde z_i)
>
s(z_i,\tilde z_j),
\qquad y_j\neq y_i.
}
\]

即正样本相似度应当高于所有异类负样本相似度。

公式中的 \(1\) 对应正样本归一化项，不能删除。若删除，损失将不再是规范的非负对比损失。

## 训练与数值计算

- 使用对称损失，使原图和增强图都作为一次锚点。
- 使用 `logsumexp` 计算损失，避免直接计算大量指数。
- 数据集全局类别均衡，因此不使用逆频率或类别数量权重。
- mini-batch 至少包含两个类别；若没有有效负样本，则该批次令 \(L_{\mathrm{CON}}=0\)。
- 默认设置：

\[
\tau=0.05,\qquad
\lambda_{\mathrm{CON}}=0.1.
\]

## 验证标准

- 同类别其他实例对应的 \(m_{ij}=0\)，不会被对比损失推远。
- 不同类别实例对应的 \(m_{ij}=1\)，作为有效负样本。
- 提高正样本相似度 \(s_i^+\) 时，损失下降。
- 提高任意异类负样本相似度时，损失上升。
- 所有损失值均非负。
- 原始形式与相对相似度形式的计算结果一致。
- 单类别批次、零向量和极端相似度不会产生 NaN 或无穷大。

## 最终解释

该损失不是传统的无监督实例对比学习，而是利用源域标签修正负样本定义的标签引导式实例对比学习：

\[
\boxed{
\text{保持同一样本的增强一致性}
+
\text{分离不同类别}
+
\text{避免同类别误负样本排斥}.
}
\]
